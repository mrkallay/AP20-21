Variable Assignment Worksheet

  1. Create a variable named x with the data type of int without assigning it a value and try to print it out (use System.out.println()).  What happens?

  1. Assign an integer value to x prior to the System.out.println() call and confirm it works.

  1. Create a variable named y with the data type of double and assign it a value in one line.

  1. Use System.out.println() to display the value of y.

  1. Create a third variable named z with the data type of int and assign it the value of x + y.  What happens?  Why do you think this is not allowed?

  1. Change the data type of z to double and see if that changes the outcome.
